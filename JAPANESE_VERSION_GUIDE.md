# 日本語版 GreenToastSoftware - 完全ガイド

## 📋 概要

GreenToastSoftwareの完全な日本語版が完成しました！ウェブサイト全体が日本語に翻訳され、新しい `ja/` フォルダに整理されています。

## 🌍 言語サポート

現在以下の言語に対応しています：
- 🇬🇧 **英語** - `index.html`
- 🇯🇵 **日本語** - `ja/index.html`
- 🇵🇹 **ポルトガル語** - 準備中
- 🇪🇸 **スペイン語** - 準備中

### 共有リソース
- CSS ファイルは共有: `../styles.css`
- JavaScript は共有: `../script.js`
- アイコン・画像は共有: `../[filename].png`

### パス参照
日本語版のファイルは相対パス `../` を使用して親フォルダのリソースを参照します：
```html
<link rel="stylesheet" href="../styles.css">
<img src="../Toast_4K_icon.png" alt="Icon">
```
