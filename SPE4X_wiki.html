SPE 4.1: Administrative Panel

SPE 4.1 Administrative Panel redefines the way you manage your computer by putting a range of powerful features at your fingertips.

From scanning for faulty drivers to a detailed and comprehensive analysis of your entire system, this dashboard is your ultimate tool to ensure your device is performing optimally.

  • Faulty driver scanning
  • Complete system analysis
  • Intuitive interface
  • Total control over device health

SPE 4.2: SPE Installer

SPE Installer is a panel that allows you to install applications from Winget and Chocolatey, but also old Windows applications such as Windows Media Center or DVD Creator.

Note: This is a PRE-BETA version — some things may change in the final release. If you have any problems or suggestions, use the Contacts page to send an email.

SPE 4.2: SPE Extras

SPE Extras is a panel that contains some extra features. In this update it includes the function of ejecting the DVD when turning off the computer.

More features are planned for upcoming releases.

Frequently Asked Questions

Quick answers about SPE 4.X

Is it safe to use SPE 4.0?

SPE has GreenGuard+, which means that SPE 4.0 has the best security against scripts and hackers!

Can SPE 4.0 install apps like Gpedit or Windows Media Center?

SPE 4.0 does not have these features, however SPE 4.1 has the Gpedit installation feature. Other old Windows applications were planned for SPE 4.2.

Is it necessary to purchase SPE 4.0?

SPE 4.X and 5.X are free programs so everyone can use them.
