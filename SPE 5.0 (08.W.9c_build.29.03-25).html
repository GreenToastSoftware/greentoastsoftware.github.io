Hello SPE INSIDERS, today we are launching SPE 5.0 (08.W.9C_build.29.03/25) PRE-BETA for all users!

News and improvements are organized in two areas: resources and corrections.Follow below the main changes of this version.

New features gradually released

This update brings a lot of bug repair and performance improvements, but also new features!

- New SPE Terminal

New functionality:

Introducing the SPE Terminal...

The SPE Terminal is designed for developers and the GreenToastSoftware testing team, where there are commands for certain functions:

> help: indicates all commands

> open: opens certain hidden dialogs and pages

> clear: clears the terminal screen

> close: closes the terminal

> exit: closes the SPE completely.

Extended support for screen readers

SPE now offers total compatibility with screen readers, making the system more accessible to everyone.
