# GreenToastSoftware - Versão em Japonês (日本語版)

## Bem-vindo à versão em japonês do site GreenToastSoftware

Esta pasta contém uma tradução completa do site para o idioma japonês, seguindo a estrutura de convenção internacional de localização.

## Estrutura de Arquivos

```
ja/
├── index.html              # Página inicial em japonês
├── about_ja.html           # Página "Sobre nós" em japonês
├── contact_ja.html         # Página de contato em japonês
└── learn/                  # Pasta para conteúdo educacional (em breve)
```

## Como Acessar

- **Página Inicial (日本語)**: `ja/index.html`
- **Sobre Nós (日本語)**: `ja/about_ja.html`
- **Contato (日本語)**: `ja/contact_ja.html`

## Características da Tradução

✅ Interface completa traduzida para japonês  
✅ Mantém toda a funcionalidade do site original  
✅ Mesmos estilos CSS (arquivo shared)  
✅ Navegação intuitiva em japonês  
✅ Formulários de contato adaptados  
✅ Suporte para múltiplos idiomas

## Arquivo de Estilos

A versão em japonês compartilha o arquivo `styles.css` da pasta raiz. Todos os caminhos para recursos estão configurados corretamente com `../` para apontar para os arquivos originais.
