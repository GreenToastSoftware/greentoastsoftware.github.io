## 🔧 Como Funciona Tecnicamente

### Compartilhamento de Recursos

A versão em japonês **compartilha** os mesmos arquivos de estilo e script:

```html
<!-- Dentro de ja/index.html -->
<link rel="stylesheet" href="../styles.css">        <!-- CSS compartilhado -->
<script src="../script.js"></script>                <!-- JavaScript compartilhado -->
<img src="../Toast_4K_icon.png" alt="Icon">        <!-- Imagens compartilhadas -->
```

✅ **Vantagens:**
- Economia de espaço em disco
- Fácil manutenção
- Consistência visual garantida
